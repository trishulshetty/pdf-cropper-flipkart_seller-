# Flipkart PDF Pro - Label & Invoice Splitter

A premium, client-side web application for Flipkart sellers to precisely split shipping labels and tax invoices from bulk PDF files.

![UI Preview](https://github.com/trishulshetty/pdf-cropper-flipkart_seller-/blob/main/preview.png?raw=true)

## 🚀 Features

- **Visual Precision**: Live preview of Page 1 with adjustable cropping boxes.
- **Vertical & Horizontal Controls**: 
  - Main horizontal split.
  - Label width adjustment.
  - Label left/top offset for pixel-perfect shipping labels.
- **Privacy First**: All processing happens in the browser. No files are uploaded to any server.
- **Bulk Processing**: Handles multi-page PDFs in seconds.
- **One-Click Download**: Automatically generates two separate PDF files: one for labels and one for invoices.

## 🛠️ Built With

- **React + Vite**
- **Tailwind CSS & Glassmorphism UI**
- **PDF-Lib**: For high-performance PDF manipulation.
- **PDF.js**: For localized rendering and preview.
- **Lucide React**: For premium iconography.

## 📦 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/trishulshetty/pdf-cropper-flipkart_seller-.git
   cd pdf-cropper-flipkart_seller-
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run locally:**
   ```bash
   npm run dev
   ```

## 🔒 Security

This app uses local processing. Your business documents never leave your computer.
